チームワークスペース

ワークスペース(Workspace)は、Aivoryの複数のコラボレーションユニットです。完全に独立会議、プロジェクト、ナレッジベースを含む共有スペースは、メンバー全員に表示され、メンバー全員が会議を継続し、実際のメッセージを送信する人の頭部にコストが記録されます。このページでは、個人のスペースとチームスペースの関係、参加と招待の作成、コンテンツの共有の可視性、メンバーの権限、スペースの切り替え、会話のアーカイブをカバーします。

個人空間とチーム空間​

それぞれのアカウントには、自然に個人のスペースがあり、追加または作成する各ワークスペースは、並行して隔離された独立したスペースです。

個人空間ワークスペース
会話 / プロジェクト / ナレッジベース自分だけが見える。メンバー全員が見られます。
コンテンツ相互ワークスペースと互換性がない空間間の相互作用もない。
ナレッジベース 吊る個人のナレッジベースだけ。この空間のナレッジベースだけを吊るす
記憶と人格普通の注入完全無注入(プライバシー保護)
会計自分の口座に誰が誰に請求するのか

隔離は双方向の硬い境界線: ワークスペース内で開かれたセッションは個人の空間リストに表示されず、個人のナレッジベースはワークスペース内のセッションに貼り付けられず、その逆です。 プロジェクトとナレッジベースの名前のユニークさは、スペース別に計算され、スペース間で名前を変更できます。

メモリは共有スペースで閉じる

シェア 会話 あなたの個人的な記憶や Persona を注入したり、シェア 会話から新しい記憶を抽出したりしない あなたの個人的な好みは同僚に漏洩されず、同僚の発言は「あなたの事実」として記録されません。口座と個性化。

オープン条件​

ユーザー グループによってワークスペースを作成できるか: ユーザー グループの編集ページで管理者が「ワークスペース」スイッチと「ワークスペースの最大数」の上限(0 が無制限)を有します。ユーザーとクォーター。

  • グループが開かれていない場合、「あなたのパッケージはワークスペースの作成をサポートしていません」というメッセージが表示されます。
  • 数値の上限に達したとき、「あなたのパッケージは作成可能なワークスペースの上限に達しました」というメッセージを表示します。
  • 招待リンクで参加他人が作成するスペースは、この制限を受けません。

ワークスペースの作成​

  1. サイドバーの下部を開きます。頭みたいなメニュー「ワークスペース→ワークスペースを作成する」を選択します。
  2. スペース名を記入し、作成を確認します。
  3. 作成が成功した後、自動的に新しいスペースに切断され、メンバーボールウィンドウに招待リンクがすぐに表示され、同僚に直接送信できます。

クリエイターは自動的にスペースの最初のメンバーとなり、キャラクターは「クリエイター」であり、残りのキャラクターはリンクを介して「メンバー」になります。

招待と参加​

招待リンク​

ワークスペース**招待リンクを介してのみ参加できます。**検索、アプリケーション、またはメールの招待メカニズムはありません。リンクは以下のとおりです。

https://your-domain.example/workspace/join/<token>

そのうちのトークンは192ビットのランダム能力トークンで、リンクを保有すれば加入できますので、信頼できる人にのみ送信してください。

最初のメニューの「メンバー」ボタンで、クリエイターは:

操作説明
招待リンクのコピー参加したい人へ
リンクをリセットする(古いリンクは即座に有効になります)新しいトークンが生成され、古いリンクは破棄され、既に加入したメンバーは影響を受けません。
メンバー削除メンバーをスペースから追い出す。
ワークスペースの削除以下の「削除とレベル連結」を参照

一般メンバーは、同じウィンドウでメンバーリストを見ることができます。ワークスペースの撤退。

流れに参加​

  1. 招待リンクを開く;ログインしていない場合は、まずログイン / 登録をジャンプし、終了後、加入ページに戻ります。
  2. ページに追加すると、スペースのプレビュー(「X によって作成された · N メンバー」)と説明が表示されます: 参加すると、スペースの共有セッション、プロジェクト、ナレッジベースにアクセスできます。
  3. 加入を確認した後、自動的に入力します。

リンクが作成者によってリセットされた場合、追加ページは「この招待リンクは有効でなくなった」というメッセージを表示し、作成者に新しいリンクを求めます。

コンテンツの共有と可視性​

ワークスペースに入ると、サイドバーの会話履歴、プロジェクトリスト、ナレッジベースのリストが表示されます。この空間のすべての内容誰が作ったかは分からない。

  • 会話: サイドバー 各 会議のタイトルの右側には、作成者の頭文字とネームが表示されます; メンバーは、すべての会議を開いて、完全な履歴を表示し、直接チャットを続けることができます。
  • バブル属自己発信のメッセージは右側にあり、他のメンバーの質問は左側にあり、著者の頭文字と名を持ち、AIの回答も左側にあります。
  • 並行質問複数のメンバーが同時に1つの会話で質問することができ、誰かの答えはまだ生成されていて、他の人の質問は間違っていないため、コンテキストの組み合わせは未完成のラウンドを自動的に飛び越えます。
  • プロジェクトとナレッジベーススペース内のプロジェクト指令、プロジェクトナレッジベース、独立ナレッジベースは、メンバー全員に有効で、メンバーの会話は、検索でアップロードできます。ナレッジベースとドキュメント質問。
  • 附属品と製品: シェア セッション でアップロードされたファイルと生成されたプロダクトは、すべてのメンバーに表示され、ダウンロードできます。

外部の公開共有リンク(ショートフォトのみ)は、空間に関係のない別のメカニズムです。共有とデータ。

支払い:誰が支払うか​

シェア セッション で生成されるすべてのクォータ、クレジット、使用量が記録されます。このニュースを送ったメンバー頭ではなく 会話 クリエイター:

  • 同僚の会話を続けることで、あなた自身の日々の割合とクレジットを消費します。
  • 回収、審査、圧縮などの衍生的任務の費用は、同様に送信者にかかる。
  • 各使用量レコードにはスペース属性があり、管理者は使用量レポートに「スペース」ロゴを表示し、スペースごとにコストを確認できます。

メンバー権限の概要​

操作宇宙の創造者普通メンバー
スペース内のすべての会話を見る / チャットを続けるできるできる
編集ニュース自分だけのニュース自分だけのニュース
会話のラウンドを削除する(質問 + 回答)自分で作成した会話で任意のラウンドを削除できます自分自身の質問を削除するだけです。
会話全体を削除自分で作った会話自分で作った会話
会話のアーカイブできるできる
招待リンクのコピー / リセットできるできない
メンバー削除できるできない
スペースからの撤退不適用できる
ワークスペースの削除できるできない

削除の入力は権限によって表示されます: 削除の権限がないメッセージには削除ボタンが表示されず、サーバー側も2度目の検証を行います。

削除とレベル​

クリエイターが ワークスペース全体を削除すると、すべてのコンテンツレベル削除: すべてのセッション、アイテム、ナレッジベース(ドキュメントとベータを含む)が同時に削除され、すべてのメンバーが自動的に終了します。この操作は復元できませんので、保存する必要のないコンテンツを確認してください。

空間交換​

3つの入力で効果が一致する:

  1. サイドバーのブランド: ワークスペースに入ると、サイドバーの上部に現在のスペース名が表示され、名前の右側には通常表示される 2 方向の矢印ボタンがあり、個人スペースに戻ります。
  2. 頭像の隣の空間スイッチ: あなたが任意のワークスペースに属する限り、サイドバーの下部の頭文字の横にスイッチボタンがあり、フラットリスト(個人スペース + あなたがいるすべてのワークスペース、現在のエントリのスイッチ)が開き、スイッチを選択します。
  3. 頭みたいなメニューまた、空間の切り替えと作成、メンバー管理の入力も含まれています。

スイッチは即時有効で、セッションリスト、アイテム、ナレッジベースが再ロードされます。現在のスペースの選択ブラウザのローカルで永続化ブラウザを閉じて再開すると、元のスペースに残り、デバイスを変更したり、ブラウザのデータを削除したりしてから個人スペースに戻ります。

会話アーカイブ​

スペース内の 会話 リストも、アーカイブをサポートしています:

  • サイドバーの会話の「⋯」メニューで「アーカイブ」を選択し、会話をリストから取り出します。
  • サイドバーで「アーカイブされたセッション」リストを開いて、表示、再開、アーカイブをキャンセルしたり(自分で作成したセッションの場合)削除したりできます。
  • アーカイブをキャンセルすると、会話は通常のリストに戻ります。

アーカイブはリストの表示に影響を与え、内容は削除されず、管理者はアーカイブを含むすべてのセッションをバックグラウンドで表示できます。

管理者の視点​

管理コンソール 独立した「ワークスペース」メニューがあります:

  • すべてのスペースの名前、作成者、メンバー数、作成時刻を表示するリスト。
  • スペースのメンバー、会話、プロジェクト、ナレッジベースを掘り起こすことができます。
  • 会話 リストは直接開いて、管理者セッションビューアで完全なチャットレコードを読み取ることができます。
  • 管理者は違反スペースを削除することができます。

よくある質問​

  • **スペースに加入する費用はかかりますか?**参加自体は、いかなる割引も消費しません;その後、あなたは「誰が請求する」を押す「誰が送信する」のスペースでメッセージを送信し、自分の割引とクレジットを消費します。
  • **招待リンクが漏れたらどうする?**クリエイターは直ちにメンバービューウィンドウで「リンクをリセット」し、古いリンクを無効にし、混ぜたメンバーは「メンバーを削除」で飛び出します。
  • **なぜ私自身が宇宙で自分の個人ナレッジベースを見ることができないのか?**これは意図的な設計:空間と個人コンテンツの両方隔離、共有する必要のある情報は、空間内で新しく作成されたナレッジベースにアップロードしてください。
  • **メンバーは私の会話を削除できますか?**会話全体 会話の作成者だけが削除することができ、メンバーは自分たちが始めた質問の数回を削除するしかありません。
