Documents et espace de stockage

La page « Documents » est une entrée de gestion unique pour tout le contenu que vous téléchargez : barre latérale, cliquez sur « Documents » en dessous de « Dessin ». Les pièces jointes de conversation et les documents de la base de connaissances sont répertoriés sur cette page – vous n’avez plus besoin de chercher une par une dans chaque conversation ou base de connaissances.

La page est constituée​

La partie supérieure est le stockage.: Affichage de l'espace utilisé et du quota total (configuré par l'administrateur en fonction du groupe d'utilisateurs) lorsque la barre de progression devient rouge lorsqu'elle dépasse 90 %. Si votre groupe n'a pas de quota, l'affichage « xx utilisé · pas de limite ».

Ci-dessous la liste des documents. Chaque ligne comprend :

列Expliquer
Nom du documentAvec l'icône de type, la suspension affiche le nom complet
TypesNom de l'extension (PDF / DOCX / XLSX / MD / PNG...)
sourceLe logo « conversation » ou le logo de base de connaissances
tailleLe nombre de documents
Temps de téléchargementTemps de localisation
opérationnelVoir en ligne / Supprimer

Prise en charge de la recherche par nom de fichier (sans distinction entre les grandes et les petites lettres), du filtrage par source (conversation / base de connaissances), de l'ordre par heure ou par taille, et du chargement par page de la liste.

Prévisions en ligne​

Cliquez sur Afficher en direct sans télécharger : affichage direct des images, lecteur PDF intégré, affichage de texte/code de haute luminosité (plus de 2 Mo de texte recommandé après téléchargement) et téléchargement dans d’autres formats.

Règles de stockage des quotas​

  • Seuls les documents de catégorie documentation sont inclus dans les quotas PDF, Word, tableaux, textes, etc. Les photos ne comptent pas Plus d’images ne prennent pas de place dans la conversation.
  • Le même byte téléchargé partage le même byte entre conversation et index de récupération, une seule fois.
  • Lorsque l'espace est plein, le téléchargement du document sera refusé, avec l'indication "L'espace de stockage est plein, allez d'abord à la page de nettoyage du fichier".
  • La taille de quota est définie par l'administrateur par groupe dans la console d'administration > groupe d'utilisateurs (0 signifie non-limité).

Supprimer les documents​

Une fois le document supprimé et confirmé, Il a été complètement supprimé de tous les endroits. L'objet du fichier sur le disque, l'enregistrement de la base de données, le vecteur de récupération sont éliminés et ne peuvent pas être récupérés.

L’erreur n’a pas été signalée après la suppression de conversation, mais deux points sont à noter:

  • Les étiquettes d'annexe dans les messages historiques s'affichent toujours, mais le fichier d'indication de prévisualisation n'existe pas; le raccourci de l'annexe d'image devient un emplacement "le fichier a été supprimé".
  • Le modèle « mémoire »: Lorsque la conversation se poursuit, le modèle ne peut plus lire le contenu de ce fichier (l'image n'est plus disponible pour le modèle visuel, le document n'apparaît plus dans les résultats de recherche), mais uniquement sur la base du texte de réponse généré par le tour précédent. base de connaissances symbolique du document - la conversation qui a téléchargé la bibliothèque après la suppression ne peut pas le récupérer, le reste du document n'est pas affecté.
