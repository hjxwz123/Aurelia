Variable environnementale optimale

Généralement pas de modification.

Les comportements internes des systèmes de contrôle des variables sont répertoriés sur cette page (parallèlement, réessai, dépassement du temps, taille du lot, etc.). Toutes les valeurs par défaut sont utilisées lorsqu'elles ne sont pas configurées et le comportement est entièrement conforme à la version officielle. Les modifications ne sont effectuées que lorsque l'impact est clairement compris; les variables de déploiement de base sont visibles. Variables environnementales clés。
