團隊工作空間

工作空間(Workspace)是 Aivory 的多人協作單元:一個完全獨立的共享空間,其中的對話、專案、知識庫對全體成員可見,任何成員都能續聊任何對話,而費用記在實際傳送訊息的人頭上。本頁覆蓋個人空間與團隊空間的關係、建立與邀請加入、共享內容的可見性、成員許可權、空間切換與對話歸檔。

個人空間與團隊空間​

每個賬號天然擁有一個個人空間;加入或建立的每個工作空間都是與之平行、互相隔離的獨立空間:

個人空間工作空間
對話 / 專案 / 知識庫只有你自己可見全體成員可見
內容互通與任何工作空間不互通空間之間也互不互通
知識庫掛載只能掛個人知識庫只能掛該空間的知識庫
個人記憶與 Persona正常注入完全不注入(隱私保護)
計費記自己賬上誰傳送誰計費

隔離是雙向的硬邊界:在工作空間裡開的對話不會出現在個人空間列表裡,個人知識庫也無法在工作空間的對話裡掛載,反之亦然。專案和知識庫的名稱唯一性也按空間獨立計算,跨空間可以重名。

記憶在共享空間中關閉

共享對話不注入你的個人記憶和 Persona,也不會從共享對話中提取新記憶。你的私人偏好不會洩露給同事,同事的發言也不會被記成「你的事實」。記憶功能本身見賬號與個性化。

開通條件​

能否建立工作空間由使用者組控制:管理員在使用者組編輯頁有「工作空間」開關與「最多工作空間數」上限(0 表示不限),見使用者與配額。

  • 所在組未開啟時,建立入口會提示「你的套餐不支援建立工作空間」;
  • 達到數量上限時提示「你的套餐已達到可建立工作空間上限」;
  • 通過邀請連結加入別人建立的空間不受此限制。

建立工作空間​

  1. 開啟側欄底部的頭像選單,選「工作空間 → 建立工作空間」;
  2. 填寫空間名稱,確認建立;
  3. 建立成功後自動切入新空間,成員彈窗立即顯示邀請連結,可直接複製發給同伴。

建立者自動成為空間的第一名成員,角色為「建立者」;其餘通過連結加入的人角色為「成員」。

邀請與加入​

邀請連結​

工作空間只能通過邀請連結加入,沒有搜尋、申請或郵箱邀請機制。連結形如:

https://your-domain.example/workspace/join/<token>

其中 token 是一段 192 位隨機能力令牌,持有連結即可加入,請只發給可信的人。

在頭像選單的「成員」彈窗中,建立者可以:

操作說明
複製邀請連結發給要加入的人
重置連結(舊連結立即失效)生成新 token,已外洩的舊連結作廢;已加入的成員不受影響
移除成員把某個成員踢出空間
刪除工作空間見下文「刪除與級聯」

普通成員在同一彈窗裡可以檢視成員列表、退出工作空間。

加入流程​

  1. 開啟邀請連結;未登入會先跳轉登入 / 註冊,完成後回到加入頁;
  2. 加入頁顯示空間預覽(「由 X 建立 · N 名成員」)和說明:加入後即可訪問該空間的共享對話、專案和知識庫,你的個人空間保持獨立;
  3. 確認加入後自動進入該空間。

連結已被建立者重置時,加入頁會提示「該邀請連結已失效」,向建立者索取新連結即可。

共享內容與可見性​

進入工作空間後,側欄的對話歷史、專案列表、知識庫列表展示的都是該空間的全部內容,不區分是誰建立的:

  • 對話:側欄每條對話的標題右側顯示建立者頭像與暱稱;任何成員都可以點開任何對話檢視完整歷史,並直接續聊。
  • 氣泡歸屬:自己發的訊息在右側;其他成員的提問在左側並帶作者頭像與暱稱,AI 回覆也在左側。對話概述樹的提問節點同樣顯示真實作者。
  • 併發提問:多名成員可以同時在一條對話裡提問;某人的回答還在生成時,其他人的提問不會因此報錯,上下文組裝會自動跳過尚未完成的回合。
  • 專案與知識庫:空間內的專案指令、專案知識庫、獨立知識庫對全體成員生效,任何成員的對話都能掛載檢索,見知識庫與文件問答。
  • 附件與產物:共享對話中的上傳檔案和生成產物對所有成員可見、可下載。

對外的公開分享連結(只讀快照)是與空間無關的另一套機制,見分享與資料。

計費:誰傳送誰計費​

共享對話中的每一次生成,配額、積分、用量都記在傳送這條訊息的成員頭上,而不是對話建立者:

  • 你在同事開的對話裡續聊,消耗的是你自己的每日額度與積分;
  • 檢索、審校、壓縮等衍生任務的費用同樣歸傳送者;
  • 每條用量記錄帶空間歸屬,管理員在用量報表中能看到「空間」徽標,按空間核對成本。

成員許可權一覽​

操作空間建立者普通成員
檢視 / 續聊空間內任何對話可以可以
編輯訊息僅自己發的訊息僅自己發的訊息
刪除一輪對話(提問 + 回答)自己建立的對話內可刪任意回合只能刪自己提問的那一輪
刪除整條對話僅自己建立的對話僅自己建立的對話
歸檔對話可以可以
複製 / 重置邀請連結可以不可以
移除成員可以不可以
退出空間不適用可以
刪除工作空間可以不可以

刪除入口按許可權顯示:沒有許可權刪除的訊息不會出現刪除按鈕,服務端同樣做二次校驗。

刪除與級聯​

建立者刪除整個工作空間時,空間內全部內容級聯清除:所有對話、專案、知識庫(含文件與向量)一併刪除,所有成員自動退出。此操作不可恢復,請先確認沒有需要保留的內容。

空間切換​

三個入口,效果一致:

  1. 側欄品牌位:進入工作空間後,側欄頂部顯示當前空間名;名稱右側有一個常顯的雙向箭頭按鈕,一鍵切回個人空間。
  2. 頭像旁的空間切換器:只要你屬於任一工作空間,側欄底部頭像旁就有切換按鈕,點開是扁平列表(個人空間 + 你所在的全部工作空間,當前項打勾),點選即切換。
  3. 頭像選單:同樣包含空間切換與建立、成員管理入口。

切換即時生效,會話列表、專案、知識庫隨之過載。當前空間的選擇持久化在瀏覽器本地,關閉瀏覽器再開啟仍停留在原空間;換裝置或清除瀏覽器資料後回到個人空間。

對話歸檔​

空間內的對話列表同樣支援歸檔整理:

  • 側欄對話的「⋯」選單選「歸檔」,對話從列表中收起;
  • 側欄可開啟「已歸檔對話」列表,檢視、重新開啟、取消歸檔,或(若是自己建立的對話)刪除;
  • 取消歸檔後對話回到正常列表。

歸檔隻影響列表展示,不刪除任何內容;管理員在後臺仍能檢視包含已歸檔在內的全部會話。

管理員視角​

管理後臺有獨立的「工作空間」選單:

  • 列表展示所有空間的名稱、建立者、成員數、建立時間;
  • 點進詳情可鑽取該空間的成員、對話、專案、知識庫;
  • 對話列表可直接點開,進入管理員會話檢視器閱讀完整聊天記錄;
  • 管理員可刪除違規空間。

常見問題​

  • 加入空間要付費嗎? 加入本身不消耗任何額度;之後你在空間裡發訊息按「誰傳送誰計費」消耗你自己的額度與積分。
  • 邀請連結洩露了怎麼辦? 建立者立即在成員彈窗裡「重置連結」,舊連結作廢;已混入的人用「移除成員」踢出。
  • 為什麼我在空間裡看不到自己的個人知識庫? 這是刻意設計:空間與個人內容雙向隔離,需要共享的資料請在空間內新建知識庫上傳。
  • 成員能刪掉我的對話嗎? 不能。整條對話只有它的建立者能刪;成員最多隻能刪除自己發起的那幾輪問答。
