文件清单

「数据 > 文件」是全站上传文件的统一视图:所有用户在对话与知识库中上传的文件,一张表看全、一处清理。适合做存储审计、清理占空间的大文件、排查违规内容。

列表与筛选​

每行显示:文件名、所属用户(邮箱)、来源(「对话」徽标或知识库名徽标)、大小、上传时间。

筛选与排序:

  • 文件名搜索:子串匹配,不区分大小写;
  • 用户:下拉选择,只看某个用户的文件;
  • 来源:全部 / 对话 / 知识库;
  • 排序:按上传时间或文件大小,双向。

同一次对话上传在数据库里会同时存在附件记录与检索文档记录(同一份物理字节),列表已自动折叠为一行,不会重复显示。

查看内容​

每行的「查看」按钮在线预览文件内容:图片、PDF、文本直接展示,其他格式可下载。用于人工审查用户上传内容。

删除与批量删除​

  • 单行「删除」或勾选多行后点「删除所选」,确认弹窗会列出将被删除的文件名。
  • 删除是彻底的三层清理:数据库记录(含检索分块)→ 向量库中的向量 → 磁盘上的文件本体。与用户侧删除走同一条链路,不会留下任何残留。
  • 批量删除时已不存在的条目自动跳过,不会导致整批失败。
与存储配额配合

在用户组中为每组配置「存储空间(MB)」后,用户的非图片上传总量将受限;这里的文件清单可以帮你找出占空间最多的用户与文件。
