团队工作空间

工作空间(Workspace)是 Aivory 的多人协作单元:一个完全独立的共享空间,其中的对话、项目、知识库对全体成员可见,任何成员都能续聊任何对话,而费用记在实际发送消息的人头上。本页覆盖个人空间与团队空间的关系、创建与邀请加入、共享内容的可见性、成员权限、空间切换与对话归档。

个人空间与团队空间​

每个账号天然拥有一个个人空间;加入或创建的每个工作空间都是与之平行、互相隔离的独立空间:

个人空间工作空间
对话 / 项目 / 知识库只有你自己可见全体成员可见
内容互通与任何工作空间不互通空间之间也互不互通
知识库挂载只能挂个人知识库只能挂该空间的知识库
个人记忆与 Persona正常注入完全不注入(隐私保护)
计费记自己账上谁发送谁计费

隔离是双向的硬边界:在工作空间里开的对话不会出现在个人空间列表里,个人知识库也无法在工作空间的对话里挂载,反之亦然。项目和知识库的名称唯一性也按空间独立计算,跨空间可以重名。

记忆在共享空间中关闭

共享对话不注入你的个人记忆和 Persona,也不会从共享对话中提取新记忆。你的私人偏好不会泄露给同事,同事的发言也不会被记成「你的事实」。记忆功能本身见账号与个性化。

开通条件​

能否创建工作空间由用户组控制:管理员在用户组编辑页有「工作空间」开关与「最多工作空间数」上限(0 表示不限),见用户与配额。

  • 所在组未开启时,创建入口会提示「你的套餐不支持创建工作空间」;
  • 达到数量上限时提示「你的套餐已达到可创建工作空间上限」;
  • 通过邀请链接加入别人创建的空间不受此限制。

创建工作空间​

  1. 打开侧栏底部的头像菜单,选「工作空间 → 创建工作空间」;
  2. 填写空间名称,确认创建;
  3. 创建成功后自动切入新空间,成员弹窗立即显示邀请链接,可直接复制发给同伴。

创建者自动成为空间的第一名成员,角色为「创建者」;其余通过链接加入的人角色为「成员」。

邀请与加入​

邀请链接​

工作空间只能通过邀请链接加入,没有搜索、申请或邮箱邀请机制。链接形如:

https://your-domain.example/workspace/join/<token>

其中 token 是一段 192 位随机能力令牌,持有链接即可加入,请只发给可信的人。

在头像菜单的「成员」弹窗中,创建者可以:

操作说明
复制邀请链接发给要加入的人
重置链接(旧链接立即失效)生成新 token,已外泄的旧链接作废;已加入的成员不受影响
移除成员把某个成员踢出空间
删除工作空间见下文「删除与级联」

普通成员在同一弹窗里可以查看成员列表、退出工作空间。

加入流程​

  1. 打开邀请链接;未登录会先跳转登录 / 注册,完成后回到加入页;
  2. 加入页显示空间预览(「由 X 创建 · N 名成员」)和说明:加入后即可访问该空间的共享对话、项目和知识库,你的个人空间保持独立;
  3. 确认加入后自动进入该空间。

链接已被创建者重置时,加入页会提示「该邀请链接已失效」,向创建者索取新链接即可。

共享内容与可见性​

进入工作空间后,侧栏的对话历史、项目列表、知识库列表展示的都是该空间的全部内容,不区分是谁创建的:

  • 对话:侧栏每条对话的标题右侧显示创建者头像与昵称;任何成员都可以点开任何对话查看完整历史,并直接续聊。
  • 气泡归属:自己发的消息在右侧;其他成员的提问在左侧并带作者头像与昵称,AI 回复也在左侧。对话概述树的提问节点同样显示真实作者。
  • 并发提问:多名成员可以同时在一条对话里提问;某人的回答还在生成时,其他人的提问不会因此报错,上下文组装会自动跳过尚未完成的回合。
  • 项目与知识库:空间内的项目指令、项目知识库、独立知识库对全体成员生效,任何成员的对话都能挂载检索,见知识库与文档问答。
  • 附件与产物:共享对话中的上传文件和生成产物对所有成员可见、可下载。

对外的公开分享链接(只读快照)是与空间无关的另一套机制,见分享与数据。

计费:谁发送谁计费​

共享对话中的每一次生成,配额、积分、用量都记在发送这条消息的成员头上,而不是对话创建者:

  • 你在同事开的对话里续聊,消耗的是你自己的每日额度与积分;
  • 检索、审校、压缩等衍生任务的费用同样归发送者;
  • 每条用量记录带空间归属,管理员在用量报表中能看到「空间」徽标,按空间核对成本。

成员权限一览​

操作空间创建者普通成员
查看 / 续聊空间内任何对话可以可以
编辑消息仅自己发的消息仅自己发的消息
删除一轮对话(提问 + 回答)自己创建的对话内可删任意回合只能删自己提问的那一轮
删除整条对话仅自己创建的对话仅自己创建的对话
归档对话可以可以
复制 / 重置邀请链接可以不可以
移除成员可以不可以
退出空间不适用可以
删除工作空间可以不可以

删除入口按权限显示:没有权限删除的消息不会出现删除按钮,服务端同样做二次校验。

删除与级联​

创建者删除整个工作空间时,空间内全部内容级联清除:所有对话、项目、知识库(含文档与向量)一并删除,所有成员自动退出。此操作不可恢复,请先确认没有需要保留的内容。

空间切换​

三个入口,效果一致:

  1. 侧栏品牌位:进入工作空间后,侧栏顶部显示当前空间名;名称右侧有一个常显的双向箭头按钮,一键切回个人空间。
  2. 头像旁的空间切换器:只要你属于任一工作空间,侧栏底部头像旁就有切换按钮,点开是扁平列表(个人空间 + 你所在的全部工作空间,当前项打勾),点选即切换。
  3. 头像菜单:同样包含空间切换与创建、成员管理入口。

切换即时生效,会话列表、项目、知识库随之重载。当前空间的选择持久化在浏览器本地,关闭浏览器再打开仍停留在原空间;换设备或清除浏览器数据后回到个人空间。

对话归档​

空间内的对话列表同样支持归档整理:

  • 侧栏对话的「⋯」菜单选「归档」,对话从列表中收起;
  • 侧栏可打开「已归档对话」列表,查看、重新打开、取消归档,或(若是自己创建的对话)删除;
  • 取消归档后对话回到正常列表。

归档只影响列表展示,不删除任何内容;管理员在后台仍能查看包含已归档在内的全部会话。

管理员视角​

管理后台有独立的「工作空间」菜单:

  • 列表展示所有空间的名称、创建者、成员数、创建时间;
  • 点进详情可钻取该空间的成员、对话、项目、知识库;
  • 对话列表可直接点开,进入管理员会话查看器阅读完整聊天记录;
  • 管理员可删除违规空间。

详见管理后台概览。

常见问题​

  • 加入空间要付费吗? 加入本身不消耗任何额度;之后你在空间里发消息按「谁发送谁计费」消耗你自己的额度与积分。
  • 邀请链接泄露了怎么办? 创建者立即在成员弹窗里「重置链接」,旧链接作废;已混入的人用「移除成员」踢出。
  • 为什么我在空间里看不到自己的个人知识库? 这是刻意设计:空间与个人内容双向隔离,需要共享的资料请在空间内新建知识库上传。
  • 成员能删掉我的对话吗? 不能。整条对话只有它的创建者能删;成员最多只能删除自己发起的那几轮问答。
